
# Python botnet project

Running commands on computers from telegramchat. 




## Commands 
 
/webcam + ip/allbots

/screenshot + ip/allbots

/keylogger + ip/allbots  +  time (seconds)

/url + ip/allbots + url

/shell + ip + command (ls, cd, cd ..)

/upload + ip + downloadlink
